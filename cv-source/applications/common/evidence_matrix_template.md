# Evidence Matrix

Complete this before drafting. Do not move unsupported requirements into the resume or cover letter as if they are experience.

| Job requirement | Evidence from CV or teaching materials | Strength | Resume use | Cover letter use | Gap or caution |
| --- | --- | --- | --- | --- | --- |
|  |  | Direct / adjacent / weak / unsupported |  |  |  |

Strength definitions:

- Direct: the exact experience, method, tool, or role is supported.
- Adjacent: the source evidence is transferable, but not the exact requirement.
- Weak: only a broad skill or related context is supported.
- Unsupported: do not claim it. Ask David for evidence or list it as a gap.
