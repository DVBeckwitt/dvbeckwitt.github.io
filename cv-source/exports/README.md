# AI-readable CV exports

Generated by:

```bash
python scripts/export_cv_ai.py
```

Files:

- `Beckwitt_CV_AI.md` - preferred for AI systems and manual review.
- `Beckwitt_CV_AI.json` - structured evidence source for scripts and retrieval systems.
- `Beckwitt_CV_AI.txt` - plain-text fallback.

These exports omit private phone numbers, addresses, and professional references by default. They are not designed resumes. Use the job-family formats in `../applications/job_families/` to create targeted materials.
