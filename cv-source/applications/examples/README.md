# Application Examples Evidence Boundary

Files in `applications/examples/` are style references only. They are not evidence sources.

Do not copy any claim from an example into a new resume, cover letter, teaching statement, screening note, or interview note unless the same claim is supported by:

1. `data/cv_master.json`.
2. The current LaTeX CV source.
3. Teaching documents.
4. A user-provided update for the specific application.

If an example conflicts with `data/cv_master.json`, the example is stale. Use the current CV source instead.
